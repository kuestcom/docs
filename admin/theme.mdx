<Info>
Set your name and logo in [General](/admin/general) first. Theme works best after the brand identity is already in place.
</Info>

<AccordionGroup>
  <Accordion title="Preset" icon="swatch-book" defaultOpen>
    Start with the closest preset instead of editing every token from zero. Presets give you a balanced starting point for light and dark mode.
  </Accordion>
  <Accordion title="Corner roundness" icon="radius">
    The radius control changes how rounded buttons, cards, and inputs feel. The UI offers quick presets such as sharp, soft, and round, plus a reset action if you want to return to the default.
  </Accordion>
  <Accordion title="Core, brand, outcome, and chart colors" icon="pipette">
    The theme editor is grouped into core tokens, brand tokens, outcome colors, and chart colors. Core tokens shape readability, brand tokens affect primary actions, outcome tokens control Yes and No states, and chart tokens affect data visuals.
  </Accordion>
  <Accordion title="Live preview" icon="monitor">
    The page renders previews while you edit, including buttons, cards, and chart colors. Use this before saving so you catch contrast problems early.
  </Accordion>
</AccordionGroup>

<Warning>
Avoid low contrast, especially on muted text, borders, and Yes or No colors.
</Warning>
