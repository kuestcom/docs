---
title: Locales
description: Enable site languages and decide whether event titles and categories should be translated automatically.
icon: languages
---

<Info>
Supported locales today are `en`, `de`, `es`, `pt`, `fr`, and `zh`. English stays enabled as the default locale.
</Info>

<AccordionGroup>
  <Accordion title="Enable site languages" icon="globe" defaultOpen>
    Each switch controls whether that locale appears on the public site. English cannot be disabled. Only enable languages your team can actually review, support, and moderate.
  </Accordion>
  <Accordion title="Automatic translations" icon="sparkles">
    This switch translates event titles and category names. It does not replace human review. The feature only works when OpenRouter is configured in [General](/admin/general).
  </Accordion>
  <Accordion title="What users will notice" icon="languages">
    Enabling a locale changes the site language selector and lets translated event titles and category labels appear in that language. If translations are disabled, users still see the locale, but content stays closer to the source language.
  </Accordion>
</AccordionGroup>
