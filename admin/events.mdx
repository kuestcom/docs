---
title: Events
description: Review existing events, filter the table, manage visibility, and handle livestream or sports actions.
icon: calendar-range
---

<AccordionGroup>
  <Accordion title="Table and filters" icon="filter" defaultOpen>
    This page is for operating events that already exist. You can search by text and filter by main category, creator, series, and active-only status. The table also shows status, 24h and total volume, created date, and end date.
  </Accordion>
  <Accordion title="Visibility toggle" icon="eye-off">
    The eye icon hides or shows an event in public event lists. This does not delete the event. It only controls whether users can discover it on the site.
  </Accordion>
  <Accordion title="Livestream URL" icon="radio">
    The radio icon opens the livestream field for an event. Use it when you want the event page to link out to a live broadcast or live coverage page.
  </Accordion>
  <Accordion title="Sports final status" icon="trophy">
    Sports moneyline events get an extra trophy action. This lets the admin mark a game as ended and set the final score directly from the events table.
  </Accordion>
  <Accordion title="Sync settings" icon="settings-2">
    The settings dialog controls whether newly synced events auto-deploy or stay manual until your team reviews them. Turn auto-deploy off if you want a human review step before new synced markets go live.
  </Accordion>
  <Accordion title="Calendar and manual creation" icon="square-plus">
    The events area also links to the calendar-based creation flow for new markets. Use [Create Market](/admin/create-market) when you want the step-by-step guide for building a new event manually.
  </Accordion>
</AccordionGroup>
